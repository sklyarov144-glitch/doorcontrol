# Демо-сценарий

## Руководитель

1. Открывает `Дашборд`.
2. Видит деньги, риски, отставания, акты ОХ и замечания ТН.
3. Открывает `Центр проблем`.
4. Находит проблемную дверь или корпус.
5. Ставит задачу ИТР.
6. Открывает `План бригад`.
7. Смотрит план-факт, деньги, рейтинг бригад и рейтинг ИТР.
8. Открывает `Автоотчёт`.
9. Копирует отчёт за день для отправки руководству.

## ИТР

1. Открывает `Мои задачи`.
2. Берёт задачу в работу.
3. Добавляет комментарий.
4. Добавляет ссылку на акт или документ на Яндекс.Диске.
5. Открывает дверь из задачи.
6. Отмечает дверь смонтированной или переданной ТН.
7. Возвращается в `План бригад`.
8. Вносит факт за день.
9. Если есть отставание, выбирает причину.
10. Копирует автоотчёт.

## Что показать как ценность

- Руководитель видит состояние компании за 30 секунд.
- ИТР закрывает рабочий день без лишних таблиц.
- Просрочки превращаются в задачи и уведомления.
- Акты ОХ и замечания ТН не теряются.
- Бригады сравниваются по плану, факту и деньгам.
- Автоотчёт собирается из факта, а не пишется вручную.

