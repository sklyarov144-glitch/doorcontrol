# ГРОСС Монтаж: статусы MVP

Документ фиксирует стартовые статусы для дверей, проемов, замечаний и актов ответственного хранения.

## Статусы дверей

- Не начато - по двери еще нет выполненных работ.
- Проем готов - проем подготовлен к дальнейшим работам.
- Дверь доставлена - дверь доставлена на объект или в рабочую зону.
- Установлена - дверь смонтирована.
- Принята - дверь проверена и принята ответственным сотрудником.
- Есть замечания - по двери есть открытые замечания.
- Передана на ответственное хранение - дверь передана по акту ответственному лицу.

## Статусы проемов

- Не проверен - состояние проема еще не зафиксировано.
- Требует подготовки - проем не готов к установке двери.
- Готов - проем готов к установке двери.
- Есть отклонения - есть зафиксированные несоответствия или ограничения.
- Принят - проем проверен и принят для дальнейших работ.

## Статусы замечаний

- Открыто - замечание создано и требует обработки.
- В работе - замечание принято в работу.
- Устранено - работы по замечанию выполнены.
- Проверено - устранение проверено ответственным сотрудником.
- Закрыто - замечание полностью закрыто.
- Отменено - замечание признано неактуальным.

## Статусы актов ответственного хранения

- Не требуется - акт для двери или группы дверей не нужен.
- Требуется - акт должен быть оформлен.
- Подготовлен - данные акта внесены, но акт еще не подписан или не подтвержден.
- Подписан - акт оформлен и подписан.
- Передано на хранение - дверь или группа дверей передана ответственному лицу.
- Закрыт - обязательства по акту завершены.

## Правила по файлам и фото

- Файлы и фото допускаются только для актов ответственного хранения при необходимости.
- Для дверей, проемов и замечаний в MVP используются статусы, комментарии и история изменений без отдельного хранилища фото.
