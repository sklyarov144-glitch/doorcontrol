# DoorControl

DoorControl - внутренняя веб-система строительной компании для контроля установки дверей на объектах.

MVP фокусируется на закрытой части системы: авторизация, навигация по строительной структуре объекта и учет состояния дверей, проемов, замечаний и актов ответственного хранения.

## Цель MVP

Сделать рабочую основу системы, в которой сотрудники смогут:

- войти по логину и паролю;
- видеть список объектов;
- переходить по иерархии объект -> корпус -> секция -> этаж -> дверь;
- просматривать и менять статусы дверей;
- вести учет проемов;
- фиксировать замечания;
- хранить данные по актам ответственного хранения.

## Предполагаемые модули

- Авторизация и роли пользователей.
- Реестр объектов.
- Иерархия строительства: корпуса, секции, этажи.
- Реестр дверей и их статусы.
- Учет проемов.
- Замечания и история их устранения.
- Акты ответственного хранения.
- Отчеты и выгрузки.

## Статус проекта

Проект находится на этапе подготовки основы. Сложный код и выбор финального технологического стека будут добавлены после согласования архитектуры MVP.
